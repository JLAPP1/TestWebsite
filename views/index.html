TYPECASTING

In this lab, we learned about typecasting. Typecasting is when you convert one variable by assigning it to another and changing its variable type.

ARRAYS

Arrays are a collection of values. For example, you could store five values into an array instead of making 5 different variables/values. In this lab code, we optimized the code to use a forloop for as many days as there were and then the days + their hours.

Exit Codes

The finally lab we can look at is also a Unit 2 lab named Exit Codes. (To access more visit the Exit Code link about the lab.)

EXITCODES

A different way to check something is through the exit codes. If an exit code is 0, then the code works. If it is not, then it does not work. It is useful for working the code through the terminal.
